Данный код сортирует файлы в папке ЗАГРУЗКИ по папкам по принципу расширения.


Порядок действий:

```bash
sudo dnf install gedit
gedit sort
# Вставьте код замениф 'sas' в переменой path на ваш user_name
# Сохраните
chmod +x sort
```

теперь по команде:

```
py sort
```

Файлы в папке ЗАГРУЗКИ будут сортироваться.
